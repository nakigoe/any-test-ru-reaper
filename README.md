<h1 name="en" id="en">Any-test.ru Harvester: collect all opensource test questions, answers and dummies for further review and export into Your Moodle questions bank!</h1>

<b>The script collects questions, answers and dummies for the specified test and stores them both in `txt` format to prepare for the test (only correct answers), and in `xml` format (all answers) to import into the Moodle questions bank.</b>

To import into the Moodle questions bank the generated `xml` file, cast an eye on my other repository: <a href="https://github.com/nakigoe/moodle-bank" target="_blank">github.com/nakigoe/moodle-bank</a>

## How To Use

### Change:

  •  `test_url` and the number of questions you would like to see on one page (at the end of the url)
    
### Install Python:

  •  https://www.python.org/downloads/

### Install PIP If it Has Not been Installed With Python Automatically:

  •  https://pip.pypa.io/en/stable/installation/

### Install Libraries (please open the command line interface):

  •  Selenium `pip install selenium`

### Harvest them All!

  •  double-click on `start.bat`

<h1 name="ja" id="ja">Any-test.ru Harvester：すべてのオープンソースのテスト問題、回答、ダミーを収集し、後であなたの Moodle の問題バンクにエクスポートします！</h1>

<b>このスクリプトは、指定されたテストの問題、回答、ダミーを収集し、テストの準備（正解のみ）のために `txt` 形式で保存し、Moodle の問題バンクにインポートするために `xml` 形式（すべての回答）で保存します。</b>

Moodle の問題バンクに生成された `xml` ファイルをインポートするには、私の別のリポジトリを参照してください：<a href="https://github.com/nakigoe/moodle-bank" target="_blank">github.com/nakigoe/moodle-bank</a>


## 使い方

### 変更：

  •  `test_url` と、1 ページに表示したい質問の数（URLの末尾で）

何か他にお手伝いできることがあれば、お知らせください！
  
### Python をインストールなさってください：

  •  https://www.python.org/downloads/

### PIP が Python とともに自動的にインストールされていない場合は、PIP をインストールします:

  •  https://pip.pypa.io/en/stable/installation/

### Python ライブラリをインストールなさってください (コマンド プロンプトを開いてください)：

  •  Selenium `pip install selenium`

### すべてを収集する！

  •  `start.bat` をダブルクリックをなさってください。

<h1 name="ru" id="ru">Any-test.ru колектор: сохраните все вопросы и ответы конкретного теста для последующей подготовки к тесту и / или сохранения в Вашем банке вопросов Moodle</h1>

<b>Скрипт сохраняет вопросы и ответы конкретного теста и сохраняет их и в `txt` формате для подготовки к тесту (только правильные ответы), и в `xml` (все ответы) для последующего возможного импорта в Ваш личный банк вопросов в Moodle.</b>

Чтобы внести в Ваш банк вопросов Moodle сгенерированный скриптом `xml` файл, смотрите мой другой репозиторий: <a href="https://github.com/nakigoe/moodle-bank" target="_blank">github.com/nakigoe/moodle-bank</a>

## Инструкция

### Замените:

  •  `test_url` и количество вопросов, которое хотелось бы видеть на одной странице (цифра в конце сноски)
   
### Установите Python:

  •  https://www.python.org/downloads/

### Установите PIP, если он не установился с Python автоматически:

  •  https://pip.pypa.io/en/stable/installation/

### Установите библиотеки (откройте командную строку):

  •  Selenium `pip install selenium`
  
### Сохраняйте весь тест на здоровье!

  •  двойной щелчок мыши по `start.bat`
